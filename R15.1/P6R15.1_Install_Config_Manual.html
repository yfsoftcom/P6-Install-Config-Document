详细步骤

  1. 系统要求

    Windows Server 2008R2 Enterprise x64

    CPU Intel Xeon E5-2690 2.9GHz (2 处理器) RAM 8 GB

  2. 获取安装文件

    JDK 1.7.*

    Weblogic 10.3.6

    EPPM15.1

  3. 安装

    • JDK 1.7.*

      默认下一步即可

      设置环境变量

      创建系统变量： JAVA_HOME : 安装的JDK的目录

      创建系统变量： Classpath

      修改系统变量： Path, 添加：%JAVA_HOME%\bin

      打开控制台验证安装结果， java -version

      JDK1.7 安装完成

    • Weblogic 10.3.6

      通过命令，运行安装程序: java -jar wls1036_generic.jar

      创建新的中间件

      取消接受更新提示

      下一步

      选择安装好的jdk

      等待完成

      安装完成，不运行 QuickStart

      安装补丁

      10.3.6 的版本需要安装官方补丁，否则会导致Team Member 无法使用，本手册使用手动修改的方式安装补丁

    • EPPM15.1

      运行安装引导程序

      默认下一步

      选择需要安装的应用： P6 和 Team Member(选装Webservice)

      确认安装

      创建新的数据库链接

      

      创建新的Weblogic域

      P6配置引导程序

      创建新的P6数据库

      输入数据库链接信息

      Host Address: 127.0.01

      DBA Password: oracle

      Database Name: sws

      输入P6的系统用户信息，建议密码和用户名一致

      输入创建的域的密码：weblogic1

      完成配置，点击下一步安装

      等待安装，打开需要10分钟左右

      安装完成

      保存一下配置信息

      记录了P6和TM的链接地址

      • 配置webservice

        运行 adminpv.cmd

        输入用户名密码 privuser/privuser

        展开 P6 的 Webservice 节点

        将部分属性值设置为 false，保存即可

  4. 运行和重启P6服务

    在系统开始菜单>P6 EPPM>Start Primavera.bat 和 Stop Primavera.bat

  5. 客户端运行

    P6: http://[server-ip]:8203/p6

    Team Member: http://[server-ip]:8207/p6tmweb
